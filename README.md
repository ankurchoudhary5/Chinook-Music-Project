# Chinook Music Store — SQL Data Analysis Project

This project uses SQL queries to clean, analyze, and extract insights from the Chinook Music Store database. The focus is on understanding customer behavior, sales trends, and opportunities to improve marketing strategies.

## 🚀 Project Highlights

- **Data Cleaning:** Removed duplicates and ensured data integrity for accurate analysis.  
- **Trend Analysis:** Identified the most popular artists and albums in the USA.  
- **Genre Popularity:** Compared music genre preferences inside and outside the USA.  
- **Sales Insights:** Analyzed sales performance across different regions.  
- **Customer Behavior:**  
  - Examined purchase frequency and churned customers.  
  - Developed incentive programs to reward loyal customers and win back churned ones.  
- **Cross-Selling Analysis:**  
  - Discovered which genres are frequently bought together.  
  - Identified artists commonly purchased by the same customers.

## 🛠 Tools & Techniques

- SQL (MySQL/PostgreSQL/SQLite - specify your DB)  
- Complex SELECT queries, JOINs, GROUP BY, subqueries  
- Data cleaning and deduplication techniques  
- Analytical queries for customer segmentation and trend identification

- Documentation on insights and business recommendations  

## 💡 Purpose

This project demonstrates the power of SQL for deep business analysis and customer behavior understanding in the music retail industry. It showcases skills in data cleaning, complex querying, and deriving actionable business insights.
